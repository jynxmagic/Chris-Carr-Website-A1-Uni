Units 1 & 2

Introduction to Web Design and Development

In this course, we have completed the following work: HTML/HTML5, CSS/CSS3, Menus, Usability and Accessability & Animations.

I have been programming for the past 5 years so I have a good understanding of HTML and CSS - which is the first language most people learn.

One thing that I really liked about this unit is the introduction of animation using CSS. I have never took the time to understand CSS animation, but after doing some work with it, IT'S AWESOME.

I mainly used the Mozilla list of CSS styles and selectors when making the site, here is a link: https://developer.mozilla.org/en-US/docs/Web/CSS/Reference

Another site that I used to help me with the animation was W3Schools. Because I'm really nice, here's another link: https://www.w3schools.com/css/css3_animations.asp

I used the W3 Consortium CSS validator to make sure my code was top notch, and up to current standards. You want another link don't you? Here it is: https://jigsaw.w3.org/css-validator/

If you look at the footer of the site, you'll notice an awesome badge from W3 Consortium; the badge basically means the site is well written.

Information Systems
