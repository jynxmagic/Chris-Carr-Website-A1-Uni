Units 3 & 4

Computer System Fundamentals

Computer System Fundamentals contains 2 subject areas: Discrete Mathmatics and Electronic Engineering

Discrete Mathmatics

In Discrete Mathmatics we have covered a few areas, ranging from Sets and Matrics to Vectors and Units.

I enjoyed learning about Sets, as they remind me very much of Arrays in any language, as they are a variable containg X amount of entries.

Matrics were also fun to learn about as they were just a variation of sets.

We have just begun work on Vectors and Units, and so far the work involved is very interesting.

Maths has always been fun for me as it is almost like another language to learn.

Electronic Engineering

Electronic Engineering is probably the hardest out of all of the units that we have been doing.

It involves: creating truth tables, creating and simplifying boolean algebra, drawing circuits in logisim and creating Karnaugh Maps.

We have recently took a test for Electronic Engineering, and I'm hoping to have scored a high grade!!!

I think that the end goal of electronic engineering will be to create circuits and be able to "teach" those circuits to do something in assembly language.

Programming

Another awesome subject!

We haven't begun programming with actual Java yet, but we have been programming in something called Processing, which is a very simplified form of the java runtime environment. Here is a link to the Processing website: https://processing.org/

We have also recently took a programming test, and I'm hoping for high grades in this as well.

Hopefully, we'll be moving onto more intricate Java work soon, dealing with database connection and handing APIs.
